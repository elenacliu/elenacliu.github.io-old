    Vim 常用快捷键 #

    Vim 有三种模式，分别为 normal mode, insert mode, visual mode。

    本文仅用于记录我在日常使用中接触到的快捷键，遇到就加入，持续更新，不保证完整性。（拖了好几年终于受不了在 vim 里只能在 insert mode 中使用⬆️⬇️⬅️➡️键进行光标移动，只能用 backspace 进行删除，正式拥抱生产力工具！）

    mode 切换 #

    进入 Vim 默认为 normal mode，如果从别的 mode 进入 normal mode 需要按 esc 键。从 normal mode 进入 insert mode，按 i ；从 normal mode 进入 visual mode，按 v。

    这些 mode 的切换都通过 normal mode 作为中介。

    Normal mode #

    显示位置信息 #

    :set number：显示行号

    :set nonumber：取消行号显示

    CTRL-G：将在屏幕底部显示当前所在位置的信息。我们还可以在CTRL-G 命令之前增加数字参数，数字越大得到的信息就越详细。命令 1CTRL-G 会显示文件的全路径。命令 2CTRL-G 会同时显示缓冲区的数字标号。

    gCTRL-G：显示当前文件中的字符统计信息。包括当前行数（Line）、列数（Col）、字数（Word）、字符数（Char）和字节数（Byte）等信息。（注意这个信息不会随着你光标的移动实时更新）

    :set ruler：将在屏幕右下角显示当前所在的行和列，以及相对于整个文件的位置。

    光标移动 #

    以字符为单位移动 #

    方法
    左 h
    下 j
    上 k
    右 l

    和其他快捷键一样，可以配合数字来移动多个字符，比如 5l 向左移动 5 个字符。

    以单词为单位移动 #

    这里的前指的是左边，后指的是右边。

    前一个单词开头：b

    后一个单词开头：w

    前一个单词结尾：ge

    后一个单词结尾：e

    如果是n个单词，在命令前加上数字即可。

    以行为单位移动 #

    移动到文件头/尾 #
    方法1 方法2
    到文件头 gg :0 或者 :1
    到文件尾 shift+g（也即是 G） :$
    移动到正数/倒数第n行 #
    方法1 方法2
    到正数第n行 {n}gg，如5gg是到第5行；或者{n}G，注意 0G 是最后一行 :{n}
    到倒数第n行 :$-{n}
    移动到当前位置起上/下n行 #
    方法
    上n行 {n}j
    下n行 {n}k
    上n行的第一个非空字符 {n}-
    下n行的第一个非空字符 {n}+
    前n个段落 {n}{
    后n个段落 {n}}
    前n个句子 {n}(
    后n个句子 {n})

    实际使用中句子的定义由于过于复杂，使用较少。

    行内移动 #

    方法
    行首 0
    行尾 $
    行首非空格 ^ 或 _ 或 g^
    行尾非空格 g_

    以屏幕为单位移动 #

    方法
    当前屏的首行 H
    当前屏的尾行 L
    当前屏的中间行 M
    光标所在字符不动，将当前行移动到屏幕顶部，通常用来查看完整的下文，比如函数、类的定义 zt
    光标所在字符不动，将当前行移到屏幕中间 zz
    光标所在字符不动，将当前行移到屏幕底部 zb
    向下翻页，移动一整个屏幕 Ctrl-f
    向上翻页，移动一整个屏幕 Ctrl-b
    屏幕向下滚动一行 Ctrl-e
    屏幕向上滚动一行 Ctrl-y

    列移动 #

    移动到第 n 列：{n}|

    历史移动 #

    :jumps：查看历史

    Ctrl-o：移动到上一个（较旧的）位置

    Ctrl-i：移动到下一个（较新的）位置
